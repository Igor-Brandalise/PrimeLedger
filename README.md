# 💰 PrimeLedger

**PrimeLedger** é uma aplicação front-end para **controle e cálculo financeiro**, desenvolvida com foco em **lógica de negócio**, **organização de dados** e **visualização clara de informações financeiras**.

> 🚧 Projeto em desenvolvimento

---

## 🎯 Objetivo do Projeto

Desenvolver uma aplicação web para demonstrar:
- modelagem de dados financeiros
- controle de entradas e saídas
- cálculos automáticos de valores
- organização de informações em tabelas
- interface clara e funcional

---

## 🚀 Funcionalidades

- Registro de entradas e saídas financeiras  
- Visualização de dados em tabelas organizadas  
- Cálculo automático de totais e saldo  
- Atualização dinâmica dos valores exibidos  

---

## 🧩 Lógica e Organização

- Gerenciamento de estado para controle das movimentações  
- Separação entre lógica de cálculo e apresentação dos dados  
- Estrutura preparada para evolução contínua  

---
